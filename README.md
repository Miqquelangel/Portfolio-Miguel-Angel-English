# Portfolio Miguel Angel English
 Tensorflow Projects (currently working on Computer Vision, CNN and NLP models)
## [PROJECT 1: Estimation of medical expenses of an insurance company based on different variables.](https://github.com/Miqquelangel/Portfolio-Miguel-Angel/blob/main/Predicci%C3%B3n_de_gastos_m%C3%A9dicos.ipynb)
* Creation of a model which estimates the price of medical expenses based on age, sex, children, region, bmi and if you're a smoker.
*  Use of libraries like Matploblib, Pandas and Scikit-learn.
* More than 1000 samples of information on which to base the model.
* Use of 'MAE', 'ReLu' activation function and 'Adam' optimizer to obtain the best model.
* Architecture and graphical representation of the best model.
* Viewing predictions with a 'MAE' ≈ € 1462
